=== SOLUÇÃO ===
Criada src/strategies/dynamic.py — DynamicStrategy:
- Recebe indicators_config (lista de dicts do banco user_strategies.indicators)
- Usa IndicatorFactory.create(type, params) para instanciar cada indicador
- Parâmetros: prioriza sliderParams (valores ajustados pelo usuário) sobre parameters
- analyze(): calcula cada indicador, coleta signal (buy/sell/neutral), vota por maioria
- Confiança = ratio de concordância (ex: 3/4 indicadores bearish = 0.75)
- min_confidence=0.40 (mais permissivo que Scalping5s=0.70)
- name retorna "Scalping5s" para manter compatibilidade com trade_executor

_get_user_custom_strategy reescrito:
- Antes: mapeava params → ScalpingStrategy/TrendM1Strategy (indicadores fixos)
- Depois: instancia DynamicStrategy com indicators_config completo do banco
