[CRÍTICO] Martingale nunca resetava no win
  Causa: callback recebia placeholder {'_placeholder': True} e se auto-removia
  Fix: verificar placeholder ANTES de remover callback
  Arquivo: src/managers/tradeexecutor_manager.py

[CRÍTICO] Martingale nunca acionava em loss
  Causa: profit=0 em demo → 'draw' → simulação convertia para 'win'
  Fix: usar data.status (OrderStatus.WIN/LOSE) em vez de profit
  Arquivo: src/managers/tradeexecutor_manager.py

[ALTO] Toggle estratégia pessoal não desativava estratégia do sistema
  Causa: toggleStrategy() só atualizava estado local
  Fix: ao ligar pessoal → desliga sistema via API; ao ligar sistema → desliga pessoais
  Arquivo: aplicativo/TunesTrade2/src/screens/HomeScreen.tsx

[MÉDIO] Barra cinza no splash screen
  Causa: StatusBar translucent + container sem StatusBar.currentHeight
  Fix: remover translucent, usar totalHeight = height + StatusBar.currentHeight
  Arquivo: aplicativo/TunesTrade2/src/screens/SplashScreen.tsx
