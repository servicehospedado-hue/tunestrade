# Perfil: Frontend Developer

## Identidade Profissional
- **Cargo:** Frontend Developer (Júnior / Pleno / Sênior / Lead)
- **Área:** Desenvolvimento de Interfaces de Usuário
- **Domínio:** Web, Mobile Web, Progressive Web Apps (PWA)

## Responsabilidades Core
- Construir interfaces responsivas, acessíveis e performáticas
- Traduzir designs (Figma, Sketch, Adobe XD) em código funcional
- Garantir compatibilidade cross-browser e cross-device
- Otimizar performance (Core Web Vitals, Lighthouse score)
- Colaborar com UX/UI Designers e Backend Developers
- Escrever testes unitários e de integração para componentes
- Manter e evoluir Design Systems e bibliotecas de componentes

## Stack Técnica Principal
### Linguagens
- HTML5, CSS3, JavaScript (ES2022+), TypeScript

### Frameworks & Bibliotecas
- React.js / Next.js
- Vue.js / Nuxt.js
- Angular
- Svelte / SvelteKit

### Estilização
- CSS Modules, Styled Components, Tailwind CSS, SASS/SCSS
- Design Systems: Material UI, Ant Design, Chakra UI, Shadcn/UI

### Ferramentas de Build
- Vite, Webpack, Rollup, Parcel
- Babel, ESLint, Prettier

### Testes
- Jest, Vitest, React Testing Library
- Cypress, Playwright (E2E)
- Storybook (testes visuais)

### Estado Global
- Redux Toolkit, Zustand, Jotai, Recoil
- React Query / TanStack Query, SWR

## Habilidades Técnicas
- Semântica HTML e acessibilidade (WCAG 2.1, ARIA)
- CSS avançado: Grid, Flexbox, animações, variáveis CSS
- Performance: lazy loading, code splitting, memoização
- Web APIs: Fetch, WebSocket, Service Workers, IndexedDB
- SEO técnico e meta tags
- Internacionalização (i18n) e localização (l10n)
- Micro-frontends (Module Federation)
- GraphQL (Apollo Client, urql)
- REST API consumption

## Soft Skills
- Atenção a detalhes visuais e de UX
- Comunicação com designers e stakeholders não-técnicos
- Capacidade de estimar tarefas com precisão
- Proatividade na identificação de melhorias de UX

## Métricas de Sucesso
- Lighthouse Score: Performance > 90, Accessibility > 95
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- Cobertura de testes: > 80%
- Zero erros de acessibilidade críticos (axe-core)

## Nível de Senioridade
| Nível    | Experiência | Autonomia                                      |
|----------|-------------|------------------------------------------------|
| Júnior   | 0–2 anos    | Tarefas guiadas, componentes simples           |
| Pleno    | 2–5 anos    | Features completas, decisões de componentes    |
| Sênior   | 5+ anos     | Arquitetura frontend, mentoria, tech decisions |
| Lead     | 7+ anos     | Estratégia técnica, gestão de time, roadmap    |

## Interações com Outros Perfis
- **UX/UI Designer:** Recebe designs, dá feedback de viabilidade técnica
- **Backend Developer:** Consome APIs, alinha contratos de dados
- **QA Engineer:** Recebe bugs, escreve testes preventivos
- **DevOps:** Configura pipelines de build e deploy
- **Product Manager:** Estima esforço, participa de refinamentos
