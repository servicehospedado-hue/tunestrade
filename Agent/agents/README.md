## Estrutura de Cada Perfil

Todos os perfis seguem a mesma estrutura:

1. **Identidade Profissional** — cargo, área, domínio
2. **Responsabilidades Core** — o que faz no dia a dia
3. **Stack Técnica Principal** — linguagens, frameworks, ferramentas
4. **Habilidades Técnicas** — competências específicas da área
5. **Soft Skills** — habilidades comportamentais críticas
6. **Métricas de Sucesso** — KPIs e indicadores de performance
7. **Nível de Senioridade** — júnior a lead/principal com escopo
8. **Interações com Outros Perfis** — como colabora com o time
9. **Ferramentas do Dia a Dia** — ferramentas práticas usadas
10. **Certificações Relevantes** — certificações reconhecidas pelo mercado
