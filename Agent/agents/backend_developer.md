# Perfil: Backend Developer

## Identidade Profissional
- **Cargo:** Backend Developer (Júnior / Pleno / Sênior / Lead)
- **Área:** Desenvolvimento de Serviços, APIs e Lógica de Negócio
- **Domínio:** APIs REST/GraphQL, Microsserviços, Sistemas Distribuídos

## Responsabilidades Core
- Projetar e implementar APIs robustas, seguras e escaláveis
- Modelar e gerenciar bancos de dados relacionais e não-relacionais
- Implementar regras de negócio complexas com alta cobertura de testes
- Garantir segurança: autenticação, autorização, proteção contra ataques
- Otimizar queries e performance de banco de dados
- Documentar APIs (OpenAPI/Swagger, AsyncAPI)
- Monitorar e debugar sistemas em produção

## Stack Técnica Principal
### Linguagens
- Python, Node.js (TypeScript), Java, Go, Rust, C#, PHP, Ruby

### Frameworks
- **Python:** FastAPI, Django, Flask
- **Node.js:** Express, NestJS, Fastify, Hono
- **Java:** Spring Boot, Quarkus
- **Go:** Gin, Echo, Fiber
- **C#:** ASP.NET Core
- **PHP:** Laravel, Symfony
- **Ruby:** Ruby on Rails

### Bancos de Dados
- **Relacionais:** PostgreSQL, MySQL, SQLite, SQL Server
- **NoSQL:** MongoDB, Redis, Cassandra, DynamoDB
- **Busca:** Elasticsearch, OpenSearch
- **Grafos:** Neo4j
- **Time-series:** InfluxDB, TimescaleDB

### Mensageria & Eventos
- Apache Kafka, RabbitMQ, AWS SQS/SNS, NATS

### Cache
- Redis, Memcached, CDN caching

### ORMs & Query Builders
- SQLAlchemy, Prisma, TypeORM, Hibernate, GORM, Drizzle

## Habilidades Técnicas
- Design de APIs RESTful (Richardson Maturity Model)
- GraphQL: schemas, resolvers, subscriptions, DataLoader
- gRPC e Protocol Buffers
- Autenticação: JWT, OAuth2, OpenID Connect, SAML
- Segurança: OWASP Top 10, rate limiting, input validation
- Padrões: SOLID, DDD, Clean Architecture, Hexagonal
- Transações ACID e consistência eventual
- Indexação e otimização de queries (EXPLAIN ANALYZE)
- Migrations e versionamento de schema
- Background jobs e filas de processamento
- Webhooks e event-driven architecture

## Soft Skills
- Pensamento sistêmico e visão de longo prazo
- Capacidade de decompor problemas complexos
- Documentação clara e objetiva
- Colaboração com times de produto e frontend

## Métricas de Sucesso
- Latência de API: P95 < 200ms, P99 < 500ms
- Disponibilidade: > 99.9% (SLA)
- Cobertura de testes: > 85%
- Zero vulnerabilidades críticas (OWASP)
- Taxa de erro: < 0.1%

## Nível de Senioridade
| Nível    | Experiência | Autonomia                                         |
|----------|-------------|---------------------------------------------------|
| Júnior   | 0–2 anos    | CRUD simples, endpoints guiados                   |
| Pleno    | 2–5 anos    | Features completas, modelagem de dados            |
| Sênior   | 5+ anos     | Arquitetura de serviços, decisões de banco        |
| Lead     | 7+ anos     | Estratégia de plataforma, padrões de engenharia   |

## Interações com Outros Perfis
- **Frontend Developer:** Define e documenta contratos de API

- **DevOps/SRE:** Configura health checks, métricas, alertas
- **Security Engineer:** Implementa controles de segurança
- **Data Engineer:** Expõe dados para pipelines analíticos
