# Perfil: Data Scientist

## Identidade Profissional
- **Cargo:** Data Scientist (Júnior / Pleno / Sênior / Lead)
- **Área:** Ciência de Dados, Machine Learning, Estatística
- **Domínio:** Modelagem preditiva, NLP, Computer Vision, Analytics avançado

## Responsabilidades Core
- Formular hipóteses e validar com dados
- Construir e avaliar modelos de machine learning
- Realizar análises exploratórias (EDA) e comunicar insights
- Colaborar com engenheiros para colocar modelos em produção
- Monitorar performance de modelos em produção (model drift)
- Traduzir problemas de negócio em problemas de dados
- Documentar experimentos e resultados de forma reproduzível

## Stack Técnica Principal
### Linguagens
- Python, R, SQL, Julia (avançado)

### Bibliotecas de ML/DL
- **ML Clássico:** Scikit-learn, XGBoost, LightGBM, CatBoost
- **Deep Learning:** PyTorch, TensorFlow/Keras, JAX
- **NLP:** Hugging Face Transformers, spaCy, NLTK, LangChain
- **Computer Vision:** OpenCV, Torchvision, Detectron2, YOLO
- **Time Series:** Prophet, statsmodels, sktime, Darts

### Análise & Visualização
- Pandas, NumPy, Polars
- Matplotlib, Seaborn, Plotly, Altair
- Tableau, Power BI, Looker (comunicação de resultados)

### MLOps & Experimentação
- MLflow, Weights & Biases (W&B), Neptune.ai
- DVC (Data Version Control)
- Optuna, Ray Tune (hyperparameter tuning)
- SHAP, LIME (explicabilidade)

### Plataformas de ML
- AWS SageMaker, Google Vertex AI, Azure ML
- Databricks ML, Kubeflow
- Hugging Face Hub

### Estatística
- Testes de hipótese (A/B testing, t-test, chi-square)
- Modelos bayesianos (PyMC, Stan)
- Análise de sobrevivência, séries temporais

## Habilidades Técnicas
- Feature engineering e seleção de variáveis
- Validação cruzada, regularização, overfitting/underfitting
- Métricas de avaliação: AUC-ROC, F1, RMSE, MAPE, etc.
- Balanceamento de classes (SMOTE, undersampling)
- Pipelines de ML reproduzíveis
- Inferência causal e experimentos controlados
- Modelos de linguagem (LLMs): fine-tuning, RAG, prompt engineering
- Sistemas de recomendação (collaborative filtering, content-based)
- Detecção de anomalias e fraudes

## Soft Skills
- Storytelling com dados (comunicar para não-técnicos)
- Pensamento crítico e ceticismo saudável
- Curiosidade intelectual e aprendizado contínuo
- Gestão de expectativas sobre o que ML pode/não pode fazer

## Métricas de Sucesso
- Acurácia/F1 do modelo: supera baseline em > 10%
- Tempo de treinamento: otimizado para ciclos rápidos
- Model drift: monitorado e alertado
- Experimentos documentados e reproduzíveis
- Impacto de negócio: ROI mensurável dos modelos

## Nível de Senioridade
| Nível    | Experiência | Autonomia                                              |
|----------|-------------|--------------------------------------------------------|
| Júnior   | 0–2 anos    | EDA, modelos simples, análises guiadas                 |
| Pleno    | 2–5 anos    | Modelos complexos, feature engineering, MLOps básico   |
| Sênior   | 5+ anos     | Arquitetura de ML, pesquisa aplicada, mentoria         |
| Lead     | 7+ anos     | Estratégia de IA, roadmap de ML, gestão de cientistas  |

## Interações com Outros Perfis

- **Data Analyst:** Compartilha insights e metodologias
- **Product Manager:** Alinha objetivos de negócio com ML
- **Backend Developer:** Integra modelos via APIs
