# Perfil: Mobile Developer

## Identidade Profissional
- **Cargo:** Mobile Developer (iOS / Android / Cross-platform)
- **Área:** Desenvolvimento de Aplicativos Móveis
- **Domínio:** iOS, Android, React Native, Flutter

## Responsabilidades Core
- Desenvolver aplicativos móveis nativos ou cross-platform
- Garantir performance, fluidez e responsividade (60/120fps)
- Implementar UX/UI seguindo guidelines de cada plataforma (HIG, Material Design)
- Integrar APIs REST/GraphQL e serviços de terceiros
- Publicar e manter apps nas lojas (App Store, Google Play)
- Implementar testes automatizados (unitários, UI, E2E)
- Monitorar crashes e performance em produção

## Stack Técnica Principal
### iOS (Nativo)
- **Linguagem:** Swift, Objective-C (legado)
- **Frameworks:** SwiftUI, UIKit, Combine, Async/Await
- **Armazenamento:** Core Data, SwiftData, Realm, UserDefaults
- **Networking:** URLSession, Alamofire
- **IDE:** Xcode

### Android (Nativo)
- **Linguagem:** Kotlin, Java (legado)
- **Frameworks:** Jetpack Compose, Android Views, Coroutines, Flow
- **Armazenamento:** Room, DataStore, SharedPreferences
- **Networking:** Retrofit, OkHttp
- **IDE:** Android Studio

### Cross-Platform
- **React Native:** JavaScript/TypeScript, Expo, React Navigation
- **Flutter:** Dart, Bloc/Riverpod/Provider, GoRouter
- **Capacitor/Ionic:** Web technologies para mobile

### Serviços & Integrações
- Firebase: Auth, Firestore, FCM (push notifications), Crashlytics
- AWS Amplify, Supabase
- RevenueCat (in-app purchases)
- Stripe, PayPal (pagamentos)
- Google Maps, Mapbox (mapas)
- Biometria: Face ID, Touch ID, Fingerprint

### Testes
- **iOS:** XCTest, XCUITest, Quick/Nimble
- **Android:** JUnit, Espresso, Robolectric
- **Cross-platform:** Detox, Maestro, Appium

### CI/CD Mobile
- Fastlane (automação de build e deploy)
- Bitrise, App Center, Codemagic
- GitHub Actions com runners macOS

## Habilidades Técnicas
- Arquiteturas: MVVM, MVI, Clean Architecture, VIPER
- Gerenciamento de estado: Redux (RN), Bloc/Riverpod (Flutter), TCA (iOS)
- Deep links e Universal Links
- Offline-first e sincronização de dados
- Animações nativas e gestos complexos
- Acessibilidade: VoiceOver (iOS), TalkBack (Android)
- Performance: profiling, memory leaks, jank frames
- App Store Optimization (ASO) básico
- Segurança: certificate pinning, keychain/keystore, obfuscação

## Soft Skills
- Atenção a detalhes de UX e microinterações
- Empatia com usuários mobile (contexto de uso, conectividade)
- Comunicação com designers sobre viabilidade técnica
- Gestão de versões e compatibilidade retroativa

## Métricas de Sucesso
- Crash-free rate: > 99.5%
- App rating: > 4.5 estrelas
- Startup time: < 2s (cold start)
- Frame rate: 60fps consistente (sem jank)
- ANR rate (Android): < 0.1%
- Cobertura de testes: > 75%

| Nível    | Experiência | Autonomia                                              |

- **DevOps:** Configura pipelines de build e distribuição
- **Product Manager:** Estima esforço, participa de refinamentos
