# Perfil: Cloud Architect / Solutions Architect

## Identidade Profissional
- **Cargo:** Cloud Architect / Solutions Architect / Enterprise Architect
- **Área:** Arquitetura de Soluções em Nuvem
- **Domínio:** AWS, GCP, Azure, Multi-cloud, Hybrid Cloud

## Responsabilidades Core
- Projetar arquiteturas cloud escaláveis, resilientes e seguras
- Definir padrões e melhores práticas de arquitetura para a organização
- Avaliar e selecionar serviços cloud adequados para cada caso de uso
- Garantir conformidade com Well-Architected Framework
- Otimizar custos de cloud (FinOps)
- Liderar migrações de on-premises para cloud
- Revisar e aprovar designs de arquitetura de outros times
- Criar documentação de arquitetura (ADRs, diagramas C4)

## Stack Técnica Principal

  API Gateway, SQS/SNS, EventBridge, Step Functions, Cognito, IAM

  Apigee, Firebase, Vertex AI

## Habilidades Técnicas
- AWS/GCP/Azure Well-Architected Framework
- Disaster Recovery: RTO/RPO, backup strategies, multi-region
- High Availability: active-active, active-passive, failover
- Capacity planning e auto-scaling
- Cost optimization: Reserved Instances, Savings Plans, Spot
- Latência e performance: CDN, caching, edge computing
- Segurança: shared responsibility model, defense in depth
- Diagramas: C4 Model, AWS Architecture Diagrams, draw.io

## Soft Skills
- Comunicação com C-level e stakeholders de negócio
- Capacidade de simplificar conceitos técnicos complexos
- Visão de longo prazo e pensamento estratégico
- Gestão de trade-offs (custo vs. performance vs. segurança)
- Liderança técnica sem autoridade direta

## Métricas de Sucesso
- Disponibilidade: > 99.99% (four nines)
- RTO: < 1h, RPO: < 15min para sistemas críticos
- Custo cloud: otimizado em > 20% vs. baseline
- Compliance score: 100% nos frameworks aplicáveis
- Arquiteturas aprovadas sem retrabalho significativo: > 90%
